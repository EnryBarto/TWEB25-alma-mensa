# Creazione database
Per la creazione del database è sufficiente utilizzare il file `AlmaMensa.sql`, che contiene:

    - Tabelle
    - Indici, chiavi primarie ed esterne
    - Stored procedure
    - Trigger
    - 3 categorie base (bar, ristorante, mensa)

# Dati di demo
Per avere una demo pronta del sito, invece di utilizzare il file AlmaMensa.sql, basta utilizzare il **file unico** `Demo.sql`, contenente anche:

    - Account utente
    - Account mense, con orari piatti e menu
    - Recensioni
    - Prenotazioni
**NB:** Sottolineamo che tutte le recensioni inserite sono puramente casuali e non rispecchiano le valutazioni dei locali reali.
